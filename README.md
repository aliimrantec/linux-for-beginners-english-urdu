# Linux Basics (English + Urdu)

##  Project Overview
This repository is a beginner-friendly Linux learning guide written in both English and Urdu.

It helps new students understand Linux commands with simple explanations and real examples.

---

##  Purpose
- Help beginners learn Linux easily
- Provide bilingual (English + Urdu) understanding
- Build strong command-line foundation

---

## Topics Covered

### Navigation Commands
- pwd → current location
- ls → list files
- cd → change directory

### File Management
- cp → copy files
- mv → move/rename files
- rm → delete files
- mkdir → create folders
- touch → create files

### File Viewing
- cat → show file content
- nano → edit files

---

##  How to Use
Open each file inside folders and read explanation step by step.
