# cd Command

## English
The cd command is used to change directory.

## Urdu
cd command ek folder se doosre folder mein jane ke liye use hoti hai.

## Syntax
cd folder_name

## Examples
cd Desktop   (to enter in Desktop)                                   (Desktop k folder mai enter hony k liyy )
cd ~         (to go in home folder no matter where you present )     (ap jaha b ho ap ko home folder mai ly kar jati hy)
cd ..        (to move one folder back)                               (jis folder mai ap ho us sy sai aik folder back jany kiy liyy)
